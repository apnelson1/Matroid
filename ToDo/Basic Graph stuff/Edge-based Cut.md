The edge cut is a "vertex-based edge cut": a set of edges that separates two sets of vertices.

At the same time, vertex cut is currently defined like a "vertex-based vertex cut": a set of vertices that separates two sets of vertices.

However, "edge-based vertex cut" is also a valid concept especially related to Matroids.

So there are 4 possiblilities:
- Edge-based edge cut
- Edge-based vertex cut
- Vertex-based edge cut
- Vertex-based vertex cut

How useful are these concepts? More importantly, which one is useful enough to be worth defining?
